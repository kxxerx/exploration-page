- GitHub에는 deploy-only ZIP의 압축을 푼 내용물만 main/root에 업로드하세요. ZIP 파일 자체를 올리면 안 됩니다. 압축 파일은 홈페이지가 아니라 인간의 압축 욕망 덩어리입니다.
- standalone ZIP은 백업/전체 확인용이고, 실제 배포는 deploy-only ZIP을 쓰면 됩니다.
