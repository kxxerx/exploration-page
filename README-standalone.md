# pollution-exploration-standalone v1.17.1

기준: v1.16.2

## 변경 사항

- 업로드된 `재난 001호_07081328.hwpx` 기준으로 시나리오 JSON을 교체했습니다.
- 시나리오 표시명은 `드림랜드`로 변경했습니다.
- 파란색 지시사항 중 가능한 항목을 시나리오 효과/조건/아이템/사용 규칙으로 반영했습니다.
- 섹션 이동 없는 선택지는 참가자 동의 없이 즉시 처리되는 선택지로 표시되며, 황금색 빛무리 효과가 붙습니다.
- 오방사계반은 섹션 2에서 사용 시 `직원용 출입구로 진입한다` 선택지를 활성화합니다.
- 재난관리국/백일몽 기본 지급 아이템을 방 인벤토리에 자동 보강하는 프론트 로직을 추가했습니다.

## 적용

이번 버전은 프론트/시나리오 JSON 변경입니다. SQL 실행은 필요 없습니다.
GitHub에는 배포 전용 ZIP의 압축을 푼 내용물만 main/root에 올리세요.
배포 전용 ZIP에는 js/config.js가 포함되지 않습니다. 기존 저장소의 js/config.js를 유지하세요.
